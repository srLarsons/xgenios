PHP is a server scripting language, and is a powerful tool for making dynamic and interactive Web pages quickly.

PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP.

PHP is an acronym for "PHP Hypertext Preprocessor"

PHP is a open source scripting language and its scripts are executed on the server. PHP costs nothing, it is free to download and use. Its files can contain text, HTML, CSS, JavaScript, and PHP code.
